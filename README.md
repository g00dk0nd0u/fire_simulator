# FIRE Calculator

シンプルな **FIRE（Financial Independence, Retire Early）試算ツール** です。  
年齢・生活費・資産・積立額・年金・インフレ率をスライダーで調整すると、3つのシナリオでFIRE到達年齢を比較できます。

## できること
- 入力値をスライダーで直感的に調整できます
- **楽観 / 標準 / 保守** の3シナリオを同時比較
- FIRE到達までの資産推移をグラフ表示（Chart.js）
- 入力エラー時のバリデーション表示

## 使い方
1. このリポジトリをクローン
2. `fire.html` をブラウザで開く
3. スライダーを動かして結果を確認

> ビルドやサーバー起動は不要です（単体HTMLで動作）。

## ファイル構成
- `fire.html` : アプリ本体（UI / 計算ロジック / グラフ描画）

## 補足
- グラフ描画ライブラリとして CDN 経由で Chart.js を利用しております。
